== Description ==

**Enables the WP classic editor plugin and the old-style Edit Post screen with TinyMCE, Meta Boxes, etc. **

= Features =
* easy to install
* No need to settings in admin.

= Requirements =
It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

= Contact =

= Download Plugin =

If you want to download any plugin from your wordpress admin panel's Plugins page, then use our other plugin - [Download Plugin](https://wordpress.org/plugins/wp-classic-editor)

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Extract the zipped file and upload the folder `wp-classic-editor` to `/wp-content/plugins/` directory.
